# Eval Viewer

HTML viewer for Skill-Creator evaluation results.

## Purpose

Self-contained HTML viewer to display skill evaluation results from `evals/*.json` files in a readable, visual format.

## Usage

1. Open `viewer.html` in a browser
2. Click "Load Eval" to select a JSON file from `../evals/` directory
3. Or use the file picker to load any eval JSON

## Generating Review Data

Evaluations are generated by the Skill-Creator workflow:

```bash
python core/skills/core/skill-creator/scripts/evaluate_skill.py <skill-path>
```

This creates `evals/<skill-name>-eval.json` with:
- Overall score
- Category scores (structure, documentation, cross-platform)
- Specific feedback items
- Improvement suggestions

## File Structure

```
eval-viewer/
├── README.md          # This file
└── viewer.html        # Self-contained HTML viewer

../evals/              # Evaluation output directory
├── skill-a-eval.json
└── skill-b-eval.json
```

## Eval JSON Schema

```json
{
  "skill_name": "skill-name",
  "evaluated_at": "2026-03-11T10:00:00Z",
  "overall_score": 85,
  "max_score": 100,
  "categories": [
    {
      "name": "Structure",
      "score": 90,
      "max_score": 100,
      "items": [
        {"check": "SKILL.md exists", "passed": true, "weight": 10},
        {"check": "Valid frontmatter", "passed": true, "weight": 15}
      ]
    }
  ],
  "feedback": [
    {"type": "info", "message": "Skill follows standard structure"}
  ],
  "suggestions": [
    {"priority": "medium", "suggestion": "Add cross-platform validation"}
  ]
}
```

## Features

- Dark theme matching dashboard.html
- Score visualization with progress bars
- Category breakdown
- Filterable feedback by type
- Prioritized suggestions
- Responsive design

## Browser Compatibility

Works in modern browsers with:
- ES6+ support
- CSS Grid/Flexbox
- File API (for local file loading)
