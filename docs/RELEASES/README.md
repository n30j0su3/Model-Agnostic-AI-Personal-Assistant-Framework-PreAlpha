## Convenciones de Versionado

Usamos [Semantic Versioning](https://semver.org/lang/es/):

- **MAJOR** (X.0.0): Cambios incompatibles con versiones anteriores
- **MINOR** (0.X.0): Nuevas funcionalidades, compatibles hacia atrás
- **PATCH** (0.0.X): Correcciones de bugs, compatibles hacia atrás
- **-prealpha**: Versión en desarrollo activo, puede tener cambios frecuentes
